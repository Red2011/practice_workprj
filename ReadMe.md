## Getting Started
```PowerShell
 php bin/console app:get-ptr-records *domain_name*
```


Change the *domain_name* to the desired domain name.

## With Docker
```bash
docker-compose up --build -d
docker exec -it practice-workprj-php-fpm bash
```
